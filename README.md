# Student-Management-System
A simple Java console-based Student Management System using OOP

Project Title: Student Management System
Language: Java

Description:
A simple console-based Java application that allows the user to manage student records. 
It supports adding, viewing, searching, and removing students.

Concepts Used:
- Object-Oriented Programming (Classes, Objects, Methods)
- ArrayList for dynamic data storage
- Scanner for user input
- Loops and Switch statements for menu navigation

Features:
1. Add new students
2. View all students
3. Search for a student by ID
4. Remove a student by ID
